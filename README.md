# Facial Expression Detector

A real-time facial expression recognition system using TensorFlow.js and Flask.

## Features
- Real-time facial expression detection via webcam
- 7 emotion classes: Angry, Disgust, Fear, Happy, Sad, Surprise, Neutral
- Live confidence scores and probability distribution
- Detection history tracking
- Screenshot capture functionality
- Beautiful modern UI with responsive design

## Technology Stack
- **Backend**: Python, Flask, TensorFlow, OpenCV
- **Frontend**: HTML, CSS, JavaScript
- **Model**: CNN trained on FER2013 dataset (simulated)
